# T6-Bo2-Open-Source-Client-Fixed
BO2/T6 client source restoration project focused on fixing, documenting, and rebuilding archived Black Ops 2 client source codes including PlusOps, T6M, FourDeltaOne, and RedactedSteamBase. Includes Visual Studio 2013 setup requirements, dependency notes, build fixes, crash debugging, and compatibility restoration for modern systems.
